# VK Продукты
Приложение для стажировки в VK

## Технологический стек
- Kotlin
- Jetpack Compose
- Jetpack Navigation
- Ktor
- KotlinX Serialization
- Hilt
- Coil

## UI штучки
- Edge-to-Edge
- Material 3
- Перевод цены в привычный формат ($*price,price*)
